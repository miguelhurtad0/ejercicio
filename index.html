Git & Git hub

Concepto de Git

Git es un sistema de control de versiones distribuidas. Estos significa que cada desarrollador tiene una copia completa del historial del proyecto en su equipo local y no depende de un servidor para trabajar.

Concepto de GitHub

GitHub es una plataforma en la web que permite almacenar gestionar proyectos que utiliza git como sistema de control de versiones. Fue adquiridad por Micosoft en 2018. En este caso es la plataforma mas popular en el mundo para alojar codigo, colaborar en proyectos de sofware y gestionar versiones de manera remota.

¿Por qué usar Git y GitHub?

📜 Historial de cambios

Permite registrar cada modificación en el código, facilitando volver a versiones anteriores o revisar el progreso.

🤝 Trabajo en equipo

Colabora con otros sin conflictos gracias al uso de ramas y pull requests que permiten fusionar cambios fácilmente.

☁️ Respaldo en la nube

Tus proyectos están almacenados en línea, accesibles desde cualquier dispositivo, con seguridad y respaldo constante.

📁 Portafolio profesional

Demuestra tus habilidades compartiendo tus proyectos públicamente. Ideal para conseguir empleos o mostrar tu evolución.

⚙️ Automatización y herramientas

Integra pruebas, despliegues y seguimiento de errores desde la misma plataforma usando GitHub Actions e Issues.

🌎 Comunidad y colaboración

Contribuye en proyectos reales de código abierto, aprende de otros y forma parte de una comunidad global de desarrolladores.
